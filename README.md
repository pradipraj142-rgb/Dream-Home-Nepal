# Dream Home Nepal 🏠

A modern, responsive real estate website built with React and Tailwind CSS for buying, selling, and investing in properties across Nepal.

## Features ✨

- **Modern Navbar** - Sticky navigation with smooth scrolling
- **Hero Section** - Eye-catching banner with call-to-action buttons
- **Property Search** - Filter properties by location, type, and budget
- **Featured Properties** - Grid display of available properties
- **About Section** - Information about the platform
- **Contact Form** - Get in touch with the team
- **Responsive Design** - Works perfectly on all devices

## Tech Stack 🛠️

- React 18.2.0
- Tailwind CSS 3.3.0
- PostCSS & Autoprefixer
- React Scripts

## Getting Started 🚀

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/pradipraj142-rgb/Dream-Home-Nepal.git
cd Dream-Home-Nepal
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Build for Production 📦

```bash
npm run build
```

This builds the app for production to the `build` folder.
